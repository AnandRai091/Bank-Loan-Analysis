# Bank Loan Analysis Dashboard

## Overview

This project analyzes bank loan performance and portfolio health using SQL, Python, Excel, and Power BI.

## Tools Used

* SQL
* Python
* Excel
* Power BI

## Key Metrics

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average DTI
* Good Loan Percentage
* Bad Loan Percentage

## Dashboard Results

* Loan Applications: 38.6K
* Funded Amount: $435.8M
* Amount Received: $473.1M
* Good Loans: 86.2%
* Bad Loans: 13.8%

## Project Workflow

Business Understanding → Data Cleaning → SQL Analysis → Power BI Dashboard → Insights
